Basics

In this section we are going to learn the basics of the Computers.

ipv4 vs ipv6

An IP address (Internet Protocol Address) is a unique number given to every device connected to a network or the internet.

Types of IP Addresses-

  1. ipv4 (older and most commonly used version)
  2. ipv6 (solve the IPv4 address shortage problem.)

DNS Server

A DNS Server (Domain Name System Server) is a system that translates human-readable website names into IP addresses.

Computers Store Characters ?

Computers do not understand letters, symbols, or words directly.

They only understand binary numbers (0 and 1).

Character Sets with Character encoding.

To store characters, computers use encoding standards. These standards assign a unique number to each character.

ASCII is one of the famous character set.

Modern character set is unicode.
